# 能力 03 — 面试提纲（interview-outline）

> 对应 Agent：`interview-planner` | 模式：subagent | 优先级：P1
>
> 📖 工作空间路径、双通道入口、双通道文件录入、静默切换、混合存储原则、`export_to_html/word/pptx` 定义 → 见 [14-通用约定.md](./14-通用约定.md)。本文档仅描述本能力特有逻辑。

---

## 一、能力描述

用户提供 JD 和候选人简历，Agent 分析岗位类型，生成结构化面试提纲，支持导出 Word/HTML，HTML 版本支持内网从本机访问。

---

## 二、完整对话流程

```
用户：帮我生成面试提纲（提供候选人简历 + JD）
  ↓
interview-planner Agent：读取 JD 和简历内容
  ↓
interview-planner Agent：分析岗位类型（技术/管理/销售/运营等）
  ↓
interview-planner Agent：生成结构化面试提纲（5个模块）
  ↓
流式输出提纲内容
  ↓
用户：可以要求调整，或要求导出
  ↓
interview-planner Agent：调用 export_to_html 或 export_to_word 导出
```

## 七、本能力的差异说明

> 通用部分见 [14-通用约定.md](./14-通用约定.md)。本节仅列出面试提纲的特殊点。

### 7.1 子目录映射

| 标准位 | 本能力具体名 | 内容 |
|--------|--------------|------|
| `00_inputs/` | `00_inputs/` | 面试背景说明、審阅重点说明 |
| `01_*_raw/` | `01_jd_refs/` | 选定 JD 的快照（从 DB 拉取） |
| `02_*_parsed/` | `02_resume_refs/` | 候选人简历纯文本快照（从 `resumes` 表拉取） |
| `03_*/` | `03_outline_drafts/` | 提纲多轮调整中间版本 |
| `04_reports/` | `04_reports/` | 最终提纲（MD）+ HTML/Word/PPT 导出产物 |

### 7.2 本能力专属的存储落点

| 内容 | 存储位置 | 说明 |
|------|---------|------|
| 提纲主体（Markdown） | DB `interview_outlines` 表 + 工作空间 `04_reports/outline.md` | DB 作为可信源，文件便于面试官反复查看 |
| 问题库快照（供未来复用） | DB `interview_questions` 表（可选） | 跨候选人累计问题资产 |

### 7.3 报告导出格式选项

提纲生成后，interview-planner Agent 询问导出格式：

| 选项 | 工具 | 阶段 | 使用场景 |
|------|------|------|---------|
| HTML（记录型） | `export_to_html({ interactive: true })` | Phase 2 | 面试官边问边记，内网可访问 |
| Word | `export_to_word` | Phase 2 | 打印带到现场使用 |
| PPT（面试小组讨论） | `export_to_pptx` 配 `interview` 模板 | Phase 3 | 集体面试讨论、报告人选 |
