# 情绪与音乐规则

## 天气 → 音乐映射
- 晴天 → 明快、阳光感
- 阴天 → 独立、梦幻
- 雨天 → Jazz、Lo-fi
- 雪天 → 古典钢琴、氛围
- 大风 → Post-rock

## 时间 → 能量映射
- 早晨(6-9) → 低能量渐升，BPM 80-100
- 上午(9-12) → 中等能量，BPM 90-110
- 下午(12-15) → 稍低，BPM 85-100
- 傍晚(15-18) → 渐升，BPM 95-115
- 晚间(18-22) → 随心，BPM 80-130
- 深夜(22-6) → 极低，BPM 60-80

## 特殊规则
- 如果用户说"累了" → 播放治愈系
- 如果用户说"开心" → 播放欢快的
- 如果用户说"想哭" → 先共情，再慢慢转向温暖
- 如果连续播放同类型超过1小时 → 适当穿插变化
